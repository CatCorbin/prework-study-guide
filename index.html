Git

  • pwd:stands for print working directory and will tell you where you are
  • ls: shows a list of files and folders in your directory
  • cd: change directory
  • cd ..: moves you backward one step
  • mkdir file-name: will make a new directory with the name of your choice
  • touch file-name.txt: will create text file with the name of your choice
  • start file-name: will open file with that name
  • code .: will open Visual Studio Code (VSC)
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates new branch and switches to it
  • git add -A: adds changes to a branch
